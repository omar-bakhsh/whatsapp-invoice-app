# نظام أتمتة إرسال الفواتير عبر الواتساب (WhatsApp Invoice Automator)

نظام ويب متكامل مصمم خصيصاً لمراكز صيانة السيارات (مثل مركز متخصص مازدا) لتبسيط عملية إرسال الفواتير للعملاء. يقوم النظام بقراءة الفواتير (سواء كانت ملفات إلكترونية أو صوراً ممسوحة ضوئياً Scanner)، استخراج بيانات العميل، وإرسال الفاتورة له عبر الواتساب بشكل آلي مع رسالة مخصصة ورابط للتقييم.

## المميزات الرئيسية

*   **التعرف الضوئي (OCR):** القدرة على قراءة الفواتير الممسوحة ضوئياً أو المصورة بالجوال واستخراج النصوص منها بدقة عالية.
*   **استخراج البيانات الذكي:** استخراج تلقائي لرقم جوال العميل واسمه من داخل الفاتورة.
*   **نظام الاستبعاد الذكي:** يتجاهل النظام أرقام المركز والرقم الضريبي تلقائياً لضمان اختيار رقم العميل الصحيح.
*   **رسائل واتساب مخصصة:** إرسال رسائل ودية باسم العميل تحتوي على الفاتورة ورابط تقييم جوجل.
*   **دعم المجلدات:** إمكانية رفع مجلد كامل يحتوي على عشرات الفواتير ومعالجتها دفعة واحدة.
*   **واجهة مستخدم عصرية:** تصميم "Glassmorphism" أنيق، متجاوب، وسهل الاستخدام.

## المتطلبات الأساسية

*   نظام تشغيل: Windows أو Linux أو macOS.
*   بيئة تشغيل: [Node.js](https://nodejs.org/) (إصدار 14 أو أحدث).
*   حساب واتساب مفعل على الهاتف.

## طريقة التثبيت

1.  قم بتحميل المشروع على جهازك.
2.  افتح واجهة الأوامر (Terminal/CMD) في مجلد المشروع.
3.  قم بتثبيت المكتبات اللازمة عبر الأمر التالي:
    ```bash
    npm install
    ```

## كيفية التشغيل

1.  ابدأ تشغيل السيرفر عبر الأمر:
    ```bash
    npm start
    ```
2.  افتح المتصفح وانتقل إلى الرابط: `http://localhost:3000`.
3.  قم بمسح رمز الاستجابة السريعة (QR Code) باستخدام تطبيق الواتساب على هاتفك لربط المتصفح.
4.  ابدأ برفع الفواتير أو المجلدات فوراً.

## التقنيات المستخدمة

*   **Backend:** Node.js, Express.
*   **WhatsApp Integration:** `whatsapp-web.js`.
*   **OCR & PDF:** `Tesseract.js`, `pdf.js`, `pdf-parse`.
*   **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6).
*   **Real-time Communication:** `Socket.io`.
