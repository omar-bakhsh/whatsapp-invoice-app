# نظام أتمتة إرسال الفواتير عبر الواتساب (WhatsApp Invoice Automator)

نظام ويب متكامل مصمم لتبسيط عملية إرسال الفواتير للعملاء. يقوم النظام بقراءة الفواتير (ملفات PDF أو ممسوحة ضوئياً)، استخراج بيانات العميل، وإرسالها آلياً عبر الواتساب مع رسالة مخصصة ورابط للتقييم.

## المميزات الجديدة (v1.1)

*   **لوحة إعدادات ديناميكية:** تم إضافة واجهة تحكم تتيح للمستخدم تعديل نص الرسالة ورابط التقييم دون الحاجة لتعديل الكود.
*   **نظام القائمة السوداء (Blacklist):** إمكانية إدارة أرقام الفروع أو الأرقام المستبعدة مباشرة من لوحة الإعدادات لتجنب الإرسال غير المقصود.
*   **دعم المتغيرات:** استخدام `{{name}}` و `{{link}}` داخل الرسائل لجعلها أكثر شخصية واحترافية.

## المميزات الأساسية

*   **التعرف الضوئي (OCR):** القدرة على قراءة الفواتير الممسوحة ضوئياً أو المصورة بالجوال باستخدام `Tesseract.js`.
*   **استخراج البيانات الذكي:** استخراج تلقائي لرقم جوال العميل واسمه من داخل الفاتورة.
*   **رسائل واتساب مخصصة:** إرسال رسائل ودية واحترافية تحتوي على الفاتورة.
*   **دعم المجلدات:** إمكانية رفع مجلد كامل ومعالجة عشرات الفواتير دفعة واحدة.
*   **واجهة مستخدم عصرية:** تصميم "Glassmorphism" أنيق ومتجاوب.

## المتطلبات الأساسية

*   بيئة تشغيل: [Node.js](https://nodejs.org/) (إصدار 14 أو أحدث).
*   حساب واتساب مفعل على الهاتف.

## طريقة التثبيت والتشغيل

1.  قم بتحميل المشروع وفتح الـ Terminal في مجلده.
2.  تثبيت المكتبات:
    ```bash
    npm install
    ```
3.  التشغيل:
    ```bash
    npm start
    ```
4.  افتح `http://localhost:3000` وامسح الـ QR Code.

## ملاحظات الأمان والخصوصية

*   يتم حفظ الإعدادات في ملف `settings.json` محلياً.
*   يتم تشفير جلسات الواتساب وحفظها محلياً في مجلد `.wwebjs_auth`.
*   **ملاحظة:** تم إضافة `.gitignore` لاستثناء ملفات الجلسات والملفات المرفوعة لضمان خصوصية بياناتك عند الرفع لـ GitHub.
